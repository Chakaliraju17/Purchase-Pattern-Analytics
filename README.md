# 📊 Purchase Pattern Analytics

## 📌 Project Overview

Purchase Pattern Analytics is a data analytics project that applies Market Basket Analysis to identify products frequently purchased together. Using the Apriori algorithm, this project uncovers customer buying behavior and provides business recommendations to improve cross-selling, product placement, and promotional strategies.

---

## 🎯 Business Problem

Retail businesses generate thousands of daily transactions, making it difficult to identify hidden purchasing patterns. This project analyzes customer transactions to discover product associations that help improve sales and customer experience.

---

## 🎯 Objectives

- Clean and preprocess retail transaction data
- Perform Exploratory Data Analysis (EDA)
- Generate frequent itemsets using the Apriori Algorithm
- Create association rules
- Develop an interactive Power BI dashboard
- Provide actionable business insights

---

## 📊 Dataset

- Online Retail Dataset
- 522,064 raw records
- 514,269 cleaned records
- 21,663 transactions
